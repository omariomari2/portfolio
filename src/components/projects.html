1. Automated Call Center with AWS

AWS Lex AWS Lambda AWS Connect JavaScript

This project builds an AI-powered call center using Amazon Connect and Amazon Lex, designed to optimize customer interactions through intelligent automation. By integrating real-time call routing, conversational AI, and personalized responses, it ensures efficient service while maintaining scalability and cost-effectiveness. With advanced workflow logic and secure cloud infrastructure, businesses can enhance customer engagement and streamline operations seamlessly.

I built this system by configuring Amazon Connect to manage inbound calls, selecting DID and Toll-Free numbers for accessibility. To automate interactions, I integrated an Amazon Lex bot, designing intents that allow customers to check balances, request advisors, and navigate services effortlessly. I also developed AWS Lambda functions that pull customer data from DynamoDB, enabling personalized greetings and dynamic call routing. To ensure security, I set up IAM roles, refining access permissions for different components, making the project ready for integrating into professional workplaces.

AI-powered call center using Amazon Connect and Lex, featuring intelligent automation, real-time routing, and personalized customer interactions for efficient service delivery.

3. WVS 1.01

Python Asyncio Gemini API

WVS is a powerful yet user-friendly website vulnerability scanner designed to eliminate the confusion around security assessments. Traditional scanners overwhelm users with technical jargon—XSS, CSRF, header injection—without offering clear guidance. WVS changes that by providing plain-language explanations alongside actionable insights, ensuring that anyone, regardless of technical expertise, can understand and address vulnerabilities effectively. With both a command-line interface for developers and a clean web interface for ease of use, WVS is built for students, developers, and startup founders who need security insights they can actually act on.

I developed WVS using Python and Asyncio, ensuring an efficient, non-blocking architecture for rapid vulnerability scans. The Gemini API powers the natural language processing component, translating complex security findings into clear, human-readable explanations. This combination allows WVS to deliver fast, comprehensive scans while maintaining a user-friendly experience. Whether handling security audits or educating users on web vulnerabilities, WVS provides an accessible, actionable approach to web security.

WVS 1.01: A website vulnerability scanner that translates technical jargon into plain language, providing clear, actionable insights for both technical and non-technical users.

Unlike phone-based timers that tempt you into endless scrolling, Pomodoro Play keeps your workflow uninterrupted by living directly in your browser.
